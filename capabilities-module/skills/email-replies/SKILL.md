only draft replies to emails from real humans. Write each reply in the agent's
voice (from `PERSONA.md`). Use `[bracketed placeholders]` for unknown facts.

Save all drafts to `drafts/replies-<YYYY-MM-DD>.md`. Report how many you drafted.

**Never send email. Never mark as read. Read-only.**
